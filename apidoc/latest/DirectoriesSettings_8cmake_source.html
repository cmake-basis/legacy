2 # @file DirectoriesSettings.cmake
3 # @brief CMake variables of project directories.
4 #
5 # This file configures the project directory structure as defined by the
6 # Filesystem Hierarchy Standard for BASIS packages.
7 #
8 # @sa http://opensource.andreasschuh.com/cmake-basis/standard/fhs/
9 #
10 # The project must follow the directory structure as defined by the
11 # <tt>PROJECT_&lt;*&gt;_DIR</tt> variables.
12 #
13 # Ideally, when changing the name of one of these directories, only the
14 # directory structure of the template needs to be updated. The BASIS CMake
15 # functions should not be required to change as they are supposed to use these
16 # variables instead of the actual names. Any change of the project directory
17 # structure has to be made with care, however, and backwards compatibility to
18 # previous releases of BASIS shall be maintained. Consider the use of the
19 # @c TEMPLATE_VERSION if required.
20 #
21 # @note The documentation of the variables can be found in Directories.cmake.in.

433 # The modules of script libraries which are intended for use by external packages
434 # are installed in the respective installation directories of the particular
435 # interpreter. For example, in case of Python, the public Python modules are
436 # installed in the site-packages directory of the found PYTHON_EXECUTABLE.
437 # In particular the modules in the PROJECT_LIBRARY_DIR are intended for use
438 # by external packages. Other modules added using the basis_add_script_library()
439 # and basis_add_script() CMake functions are by default considered to be intended
440 # for internal use by the other modules and executable scripts.
441 #
442 # Note: For those interpreters of scripting languages which by themselves do
443 # not define a common installation directory for site packages, the
444 # installation directory for public modules may be identical to the
445 # one for private modules. Moreover, the user has the option to disable
446 # the installation of public modules in the system default site directories

15 #
16 # All Nightly tests which are executed within a time window of 24 hours
17 # starting at the specified start time, will replace previous submissions
18 # made within the same time frame such that for each time frame of 24 hours
19 # only one result of test executions is present on the dashboard.
20 # The execution of the Nightly tests is usually triggered automatically
21 # via a scheduled cron job that is run once every 24 hours.
22 #
23 # Use the Continuous or Experimental test models if you want to execute

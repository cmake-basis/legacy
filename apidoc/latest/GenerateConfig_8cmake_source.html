1 ##############################################################################
2 # @file GenerateConfig.cmake
3 # @brief Generates package configuration files.
4 #
5 # This CMake script configures the \<package\>Config.cmake et al. files,
6 # once for the build tree and once for the install tree. Variables with a
7 # _CONFIG suffix are replaced in the default template file by either the
8 # value for the build or the install tree, respectively.
9 #
10 # If present, this script includes the @c PROJECT_CONFIG_DIR/ConfigBuild.cmake
11 # and/or @c PROJECT_CONFIG_DIR/ConfigInstall.cmake file before configuring the
12 # Config.cmake.in template. If a file @c PROJECT_CONFIG_DIR/Config.cmake.in
13 # exists, it is used as template. Otherwise, the default template file is used.
14 #
15 # Similarly, if the file @c PROJECT_CONFIG_DIR/ConfigVersion.cmake.in exists,
16 # it is used as template for the \<package\>ConfigVersion.cmake file. The same
17 # applies to ConfigUse.cmake.in.

146  /**
147  * @brief Open new subprocess.
148  *
149  * This method creates the subprocess and returns immediately. In order to
150  * wait for the suprocess to finish, the wait() method has to be called
151  * explicitly.
152  *
153  * @param [in] cmd Command-line given as double quoted string. Arguments
154  * containing whitespaces have to be quoted using double
155  * quotes. Use a backslash (\) to escape double quotes
156  * inside an argument as well as to escape a backslash
157  * itself (required if backslash at end of double quoted
158  * argument, e.g., "this argument \\").
159  * @param [in] rm_in Mode used for redirection of stdin of subprocess.
160  * Can be either RM_NONE or RM_PIPE.
161  * @param [in] rm_out Mode used for redirection of stdout of subprocess.
162  * Can be either RM_NONE or RM_PIPE.
163  * @param [in] rm_err Mode used for redirection of stderr of subprocess.
164  * Can be either RM_NONE, RM_PIPE, or RM_STDOUT.
165  * @param [in] env Environment for the subprocess. If NULL is given, the
166  * environment of the parent process is used.
167  */
168  bool popen(const std::string& cmd,

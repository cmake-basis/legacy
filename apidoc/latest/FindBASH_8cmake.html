Detailed Description

Find BASH interpreter.

Sets the CMake variables BASH_FOUND, BASH_EXECUTABLE, BASH_VERSION_STRING, BASH_VERSION_MAJOR, BASH_VERSION_MINOR, and BASH_VERSION_PATCH.

145 /**
146  * @brief Get absolute path of executable file.
147  *
148  * This function determines the absolute file path of an executable. If no
149  * arguments are given, the absolute path of this executable is returned.
150  * If the command names a known executable build target, the absolute path to
151  * the corresonding built (and installed) executable file is returned.
152  * Otherwise, the named command is searched in the system @c PATH and its
153  * absolute path returned if found. If the executable is not found, an
154  * empty string is returned.
155  *
156  * @todo This function currently makes use of the which command implemented
157  * in Python and called as subprocess in order to search a command
158  * in the system @c PATH. This which command is part of BASIS and
159  * can also be used on Windows. However, a native C++ implementation
160  * would be desireable.
161  *
162  * @param [in] name Name of command or @c NULL.
163  *
164  * @returns Absolute path of executable or an empty string if not found.
165  * If @p name is @c NULL, the path of this executable is returned.

209  *
210  * This function is a replacement for system() on Unix and is furthermore
211  * less platform dependent. The first argument of the given command-line string
212  * is mapped to an absolute executable file using exepath() if the given first
213  * argument is a know build target name. Otherwise, the command-line is used
214  * unmodified.
215  *
216  * @param [in] cmd Command-line given as double quoted string. Arguments
217  * containing whitespaces have to be quoted using double
218  * quotes. Use a backslash (\\) to escape double quotes
219  * inside an argument as well as to escape a backslash
220  * itself (required if backslash at end of double quoted
221  * argument, e.g., "this argument \\").
222  * @param [in] quiet Turns off output of stdout of child process to stdout
223  * of parent process.
224  * @param [out] out Output stream where command output is written to.
225  * @param [in] allow_fail If true, no exception is thrown if the exit code
226  * of the child process is non-zero. Otherwise,
227  * a SubprocessException object is thrown in that case.
228  * @param [in] verbose Verbosity of output messages. Does not affect
229  * verbosity of executed command.
230  * @param [in] simulate Whether to simulate command execution only.
231  *
232  * @returns Exit code of command or -1 if subprocess creation failed.
233  *
